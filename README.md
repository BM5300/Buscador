# Buscador
Pequeño y util programa .bat creado desde un bloc de notas para salir de apuros

Este programa actua como un buscador de archivos ofrece un menu simple e intuitivo dependiendo tu selección sera el resultado que te muestre dentro de una carpeta en el escritorio llamado "encontrado"

el programa copia y pega lo buscado en la carpeta asi que puede que en tus busquedas estes llenando tu disco pero para solucionar esto esta la opcion [3] dentro del menu el cual limpia la carpeta a la cual llegan los resultados.

la forma de utilizarlo recomiendo esta:

Haces tu busqueda y terminando de usar los archivos limpiar la carpeta con el programa.
